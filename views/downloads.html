Oto-Maintenance

Un outil de maintenance puissant et simple écrit en batch. Utilisant uniquement des outils intégrés à Windows.

windows-utility

Un outil simple écrit en batch permettant d'activer Windows, à l'aide de KMS de msguides ou de mettre à niveau Windows 10/11 Home vers Windows 10/11 Entreprise.

  • Historique des versions

Admin Services

Téléchargez Admin Services pour un usage hors ligne via votre navigateur. Contient toutes les fonctionnalités du site a stocké n'importe où.

Note pour les utilisateurs Firefox : la fonction de redirection vers votre page d'accueil pourrait contenir des bogues et rediriger à l'infini. Utilisez donc Safari, Microsoft Edge ou Chrome pour une qualité optimale !

  • Historique des versions

Admin Utility

Une application utilisée pour le débogage, la modification, la réinitialisation et bien plus encore pour Admin Services.

Note : Il existe une version, dont les mises à jour sont constantes, directement sur le site d'Admin Services. Cependant, si vous avez téléchargé une ancienne version d'Admin Service, vous pouvez l'intégrée dans cette dernière.
